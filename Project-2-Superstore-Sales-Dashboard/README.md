📊 Superstore Sales Dashboard (Excel)

📌 Project Overview

This project demonstrates how Microsoft Excel can be used to perform end-to-end sales data analysis and build an interactive business dashboard. The project covers data exploration, KPI creation, Pivot Tables, Pivot Charts and dashboard design to generate actionable business insights.

🎯 Business Objective

Analyze Superstore sales data to identify:

- Sales performance
- Profitability
- Regional trends
- Product performance
- Customer behavior
- Return analysis
- Payment preferences

🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas (SUMIFS, COUNTIFS, IF, XLOOKUP/VLOOKUP)
- Interactive Dashboard

📊 Dashboard Visualizations

- Top 10 Products

- Payment Mode Distribution
- Order Status Analysis

🔍 Key Business Insights

- Identified the best-performing product categories.
- Compared sales and profit across different regions.
- Analyzed customer purchasing patterns.
- Evaluated the impact of discounts on profitability.
- Measured return rates and delivery performance.
